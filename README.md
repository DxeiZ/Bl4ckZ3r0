###Kali Linux Installation
                
----

`$ git clone https://github.com/Eratonos/Bl4ckZ3r0`

`$ cd Bl4ckZ3r0`

`$ chmod +x bl4ckz3r0.sh`

####Start Tool

`$ sudo bash bl4ckz3r0.sh`

![](https://i.ibb.co/zPbGtdH/1.png)
                
----
####Update Tool

`$ chmod +x ./update`

`$  sudo ./update`
